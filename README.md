# MC传送点记录工具

一个简单易用的 Minecraft 坐标传送点管理工具，支持快速记录、编辑和复制传送指令。

## 功能

- 支持解析多种坐标格式（`/tp x y z`、`F3+C` 复制格式等）
- 三个维度支持：主世界、下界、末地
- 单击复制传送指令，双击编辑
- 右键菜单提供更多操作
- 数据持久化存储

## 使用方法

1. 运行 `MC_TP.exe`（或通过源码运行 `teleport_tool.py`）
2. 输入坐标和名称，选择所在世界，点击"添加"
3. 点击列表中的"复制"按钮或单击行来复制传送指令

## 从源码运行

```bash
python teleport_tool.py
```

## 打包

```batch
build.bat
```

## 下载发布版

直接下载 `MC_TP.exe` 即可使用，无需 Python 环境。
